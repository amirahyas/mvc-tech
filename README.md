# CMS-Style Blog Site

This is a CMS-style blog site that allows users to create, view, update, and delete blog posts, as well as leave comments.

## Features

- User authentication (signup, login, logout)
- View homepage with existing blog posts
- Navigate to the dashboard to view and manage user's blog posts
- Create new blog posts
- Update and delete existing blog posts
- View individual blog posts with comments
- Leave comments on blog posts
- Handle session timeout for user inactivity
